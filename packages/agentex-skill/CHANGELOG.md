# Changelog

All notable changes to @agentex/skill will be documented in this file.

## [1.0.0] - 2025-01-XX

- Interactive CLI installer (`agentex-install`)

- One-command installation: `npm install -g @agentex/skill`
- Interactive platform selection
- Automatic directory detection
